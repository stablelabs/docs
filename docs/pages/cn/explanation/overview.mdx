# 概览

Stable 是一条以 USDT0 作为原生 gas 代币的 Layer 1，标准的 EVM 工具链（Solidity、Foundry、Hardhat、ethers、viem 以及 `eth_*` JSON-RPC 方法）无需任何改动即可使用。

将你的 RPC 指向 Stable 并确认链 ID：

```text
988
```

确保你已安装 [foundry](https://www.getfoundry.sh/) 以测试以下命令：

```bash
cast chain-id --rpc-url https://rpc.stable.xyz
```

有关完整的端点列表（主网和测试网），请参阅 [连接](/cn/reference/connect)。

- 钱包、委托和代理账户 → [账户](/cn/explanation/accounts-overview)。
- 转移 USDT0 或构建支付流程 → [支付](/cn/explanation/payments-overview)。
- 部署智能合约 → [合约](/cn/explanation/contracts-overview)。
- 接入 AI 编辑器或构建由代理付费的服务 → [代理结算](/cn/explanation/agent-settlement)。
- 运行全节点或归档节点、生态系统提供商或代付 gas → [基础设施](/cn/explanation/integrate-overview)。
