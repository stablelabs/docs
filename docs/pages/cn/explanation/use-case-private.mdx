# 私密转账

资金管理操作、供应商付款和工资发放等场景中，金额具有商业敏感性，不应向全世界公开。

## 问题所在

所有标准的 EVM 转账都是公开可见的。一次工资发放或供应商结算会在链上泄露关键的业务数据：谁向谁付款、付了多少、付款频率如何。竞争对手、交易方以及任何抓取网络数据的人，无需询问就能重建出工资区间、供应商定价和资金动向。

## Stable 的应对方式

- **保密转账使用零知识密码学来屏蔽转账金额**，同时为合规保持各方可审计，因此敏感数字在不牺牲审计记录的情况下保持私密。参见[保密转账](/cn/explanation/confidential-transfer)。
- **资金流向展示了保密转账在完整 USDT 生命周期中的位置**，从入金到出金。参见[资金流向](/cn/explanation/flow-of-funds)。
