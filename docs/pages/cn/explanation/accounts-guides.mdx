# 账户指南

“账户”选项卡下的所有指南、概念和参考资料，按您要执行的操作分组。

## 设置钱包

- [**创建钱包**](/cn/how-to/create-wallet)：使用 ethers.js 或 Tether WDK 生成新密钥对或从助记词恢复。
- [**代理钱包**](/cn/reference/agentic-wallets)：AI 代理的自托管钱包（它们与用户钱包有何不同）。

## 委托账户（EIP-7702）

- [**EIP-7702 概念**](/cn/explanation/eip-7702)：EIP-7702 在 Stable 上实现的功能和安全模型。
- [**账户抽象操作指南**](/cn/how-to/account-abstraction)：将 EIP-7702 应用于批量支付、消费限额和会话密钥。

## 参考

- [**EIP-7702 API**](/cn/reference/eip-7702-api)：Type-4 交易格式和授权列表。
- [**订阅和收款**](/cn/how-to/subscribe-and-collect)：将 EIP-7702 委托应用于订阅支付流程（交叉列出）。
