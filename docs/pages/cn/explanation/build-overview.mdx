# 概览

Stable 新手？请先运行[快速入门](/cn/tutorial/quick-start)。它只需五分钟，并会发送一笔测试网交易，以便标签页的其余部分可以连接到它。

## 探索

- [**账户**](/cn/explanation/accounts-overview)：创建钱包，使用 EIP-7702 授权 EOA，并为用户和代理限定会话密钥。
- [**支付**](/cn/explanation/payments-overview)：发送 USDT0，构建 P2P 和订阅流程，使用 ERC-3009 结算发票，并使用 x402 为 API 定价。
- [**合约**](/cn/explanation/contracts-overview)：部署、验证和索引 Solidity 合约，并调用 Bank / Distribution / Staking 预编译。
- [**AI 和代理**](/cn/explanation/agent-settlement)：将 MCP 服务器连接到 AI 客户端，并公开代理可以通过提示调用的 x402 付费工具。

## 从这里开始

- [**快速入门**](/cn/tutorial/quick-start)：连接到测试网，为钱包注资，并发送您的第一笔 USDT0 交易。
- [**发送您的第一笔 USDT0**](/cn/tutorial/send-usdt0)：相同余额上的原生和 ERC-20 传输，附带 TypeScript 示例。
- [**部署智能合约**](/cn/tutorial/smart-contract)：搭建 Foundry，配置 Stable，并部署 Counter 合约。
- [**Stable SDK**](/cn/explanation/sdk-overview)：使用类型化的 TypeScript 客户端在 Stable 上进行转账、桥接和兑换。
