# 概览

初次接触 Stable？请先运行[快速开始](/cn/tutorial/quick-start)。只需五分钟即可发送一笔测试网交易，让本栏其余内容有可对接的基础。

## 探索

- [**账户**](/cn/explanation/accounts-overview) — 创建钱包、使用 EIP-7702 委托 EOA，并为用户和智能体限定会话密钥范围。
- [**支付**](/cn/explanation/payments-overview) — 发送 USDT0、构建 P2P 和订阅流程、使用 ERC-3009 结算发票，并通过 x402 为 API 定价。
- [**合约**](/cn/explanation/contracts-overview) — 部署、验证和索引 Solidity 合约，并调用 Bank / Distribution / Staking 预编译合约。
- [**AI 与智能体**](/cn/explanation/agent-settlement) — 将 MCP 服务器接入 AI 客户端，并暴露智能体可通过提示词调用的 x402 付费工具。

## 从这里开始

- [**快速开始**](/cn/tutorial/quick-start) — 连接测试网、为钱包充值，并发送你的第一笔 USDT0 交易。
- [**发送你的第一笔 USDT0**](/cn/tutorial/send-usdt0) — 在同一余额上进行原生和 ERC-20 转账，附带 TypeScript 示例。
- [**部署智能合约**](/cn/tutorial/smart-contract) — 搭建 Foundry、配置 Stable，并部署 Counter 合约。
- [**Stable SDK**](/cn/explanation/sdk-overview) — 使用类型化的 TypeScript 客户端在 Stable 上进行转账、跨链桥接和兑换。
