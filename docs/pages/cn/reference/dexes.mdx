# DEX（去中心化交易所）

Stable 上的 DEX 部署，用于现货交易、流动性提供和链上路由。Stable 已列入[官方 Uniswap v3 部署列表](https://gov.uniswap.org/t/official-uniswap-v3-deployments-list/24323/13#p-58106-stable-4)：Stable 上的 Uniswap v3 合约经治理认可为规范部署，并通过 [Stable Swap](https://swap.stable.xyz) 作为默认前端进行路由。

## Uniswap v3

Stable 上规范的 Uniswap v3 部署，具有集中流动性池和标准费率档位。Stable Swap 是持续维护的默认前端；交易通过下方的合约进行路由。跨链流动性通过 LayerZero 流入。

**功能**

- 带有 v3 头寸 NFT 的集中流动性 AMM
- 标准 `SwapRouter02`、`Quoter V2` 和 `Universal Router` 集成路径
- 支持 `Permit2` 实现无 gas 授权
- 同时部署了 v2 风格的恒定乘积池以支持旧版路由

### 获取交易报价

`Quoter V2` 在不执行交易的情况下返回给定输入的预期输出。可从任何指向 Stable RPC 的 EVM 工具中使用它。

```bash
cast call 0xb070179E7032CdA868b53e6C1742F80c9e940d1A \
  "quoteExactInputSingle((address,address,uint256,uint24,uint160))(uint256,uint160,uint32,uint256)" \
  "(<TOKEN_IN>,<TOKEN_OUT>,<AMOUNT_IN>,<FEE>,0)" \
  --rpc-url https://rpc.stable.xyz
```

```text
(amountOut, sqrtPriceX96After, initializedTicksCrossed, gasEstimate)
```

将 `<TOKEN_IN>`、`<TOKEN_OUT>`、`<AMOUNT_IN>` 和 `<FEE>`（`100`、`500`、`3000`、`10000` 之一）替换为你要报价的池的值。对于应用集成，建议使用 [Uniswap v3 SDK](https://docs.uniswap.org/sdk/v3/overview) 或指向上述地址的 [Universal Router](https://docs.uniswap.org/contracts/universal-router/overview)。
