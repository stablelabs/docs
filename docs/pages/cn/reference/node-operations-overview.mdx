## 节点类型

### 全节点
全节点维护区块链的完整副本，并验证所有交易和区块。全节点：
- 验证所有交易和区块
- 维护整个区块链历史
- 可以向其他节点提供数据
- 支持网络的去中心化

### 归档节点
归档节点存储所有状态的完整历史，并能提供历史查询。归档节点：
- 存储所有历史状态
- 支持在任意区块高度进行历史查询
- 需要显著更多的存储空间
- 对区块浏览器和分析工具至关重要

## 快速开始

面向希望快速上手的经验丰富的运维人员：

1. 查看 [系统要求](/cn/reference/node-system-requirements)
